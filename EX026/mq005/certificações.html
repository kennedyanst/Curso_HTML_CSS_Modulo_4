Análise e Ciências de Dados

Python | R | Excel | SQL

Como Analista de Dados, combino minha sólida vocação em Análise de Dados com conhecimentos em Ciência de Dados, com foco especial em algoritmos de classificação e regressão. Minha trajetória profissional me permite extrair insights valiosos a partir de dados e desenvolver modelos preditivos. Em um mundo cada vez mais orientado por dados, a capacidade de analisar informações é crucial. Minha expertise em Análise de Dados me permite transformar dados brutos em insights estratégicos para apoiar a tomada de decisões.

Análise de Business Intelligence

Microsoft Power BI | FIGMA

Sou Analista de Business Intelligence, focado em Power BI, ETL, DAX e M, com destaque na criação de dashboards e relatórios que convertem dados complexos em insights acionáveis. Minha expertise em BI ajuda organizações a tomar decisões informadas com base em informações precisas e em tempo real. Além disso, desenvolvo layouts no Figma para apresentações de dados precisas e impactantes, proporcionando uma visão clara das análises. Com a capacidade de extrair, transformar e carregar dados de várias fontes, crio processos ETL eficientes que garantem dados limpos e confiáveis para análise.

Desenvolvimento Front-end

HTML5 | CSS3 | JavaScript

Iniciei minha carreira como Desenvolvedor Frontend com a motivação de criar relatórios visualmente agradáveis, inspirado pelo meu trabalho com o Power BI. Possuo um conhecimento significativo em HTML5, CSS3 e um pouco menos JavaScript, permitindo-me criar interfaces web de alta qualidade. Minha transição do Power BI para o desenvolvimento frontend me permitiu traduzir minha paixão por visualização de dados em interfaces web atraentes. Estou comprometido em criar experiências digitais envolventes e valiosas para projetos e negócios em constante evolução no cenário digital.
